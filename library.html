Library for everyone

Kulwant Kaur Virdi Library is set up in fond memory of Late Kulwant Kaur, ex-wife Gursharan Singh Virdi. This library was started to help students which don't have access to books. Another contributor to the library is Punjabi Sahitya Sabha, New Delhi. This library is still growing; books are added from time to time as the stock becomes available.
